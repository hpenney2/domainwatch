# domainwatch

Watches a domain's RDAP info for changes.  
Notifications can also be sent via a Discord webhook, optionally specified through the `DISCORD_WEBHOOK_URL` environment variable.

<small>Please don't use this nefariously. :(</small>

> [!CAUTION]
> Check the TOS of the RDAP server for the domain that you are watching.  
> High volume requests may be against them!
>
> I am not responsible for anything that is done with program.
